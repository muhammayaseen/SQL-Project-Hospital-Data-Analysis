# 🏥 Hospital Data Analysis (SQL Project)

### 📘 Project Overview

This project is part of my **30-Day SQL Micro Course**, where I applied SQL techniques to analyze a hospital dataset. The goal was to extract key insights about patient trends, hospital efficiency, and medical expenses using structured queries.

---

### 🎯 Objectives

* Build and manage a relational database in PostgreSQL
* Clean and convert inconsistent date formats
* Write SQL queries to answer real-world analytical questions
* Generate insights for decision-making and performance evaluation

---

### 🗂️ Dataset Description

The dataset includes:

* **Hospital_Name** — Name of the hospital
* **Location** — City where the hospital is located
* **Department** — Medical department (e.g., Oncology, ENT, Pediatrics)
* **Doctors_Count** — Number of doctors in the department
* **Patients_Count** — Number of patients treated
* **Admission_Date / Discharge_Date** — Patient admission and discharge dates
* **Medical_Expenses** — Total expenses incurred

---

### 🧠 Key Insights Extracted

* 🧾 **Total Patients:** Calculated across all hospitals
* 🏥 **Average Doctors per Hospital:** Evaluated resource distribution
* 🧩 **Top Departments:** Identified highest patient loads
* 💰 **Hospital with Maximum Expenses:** Determined high-cost facilities
* 📅 **Monthly Expense Report:** Tracked expense trends over time
* ⏱️ **Average Stay Length:** Measured patient hospitalization duration
* 🌆 **Patients per City:** Compared regional healthcare activity

---

### ⚙️ Tools & Technologies

* **Database:** PostgreSQL
* **Language:** SQL
* **Platform:** pgAdmin4

---

### 🧩 Learning Outcomes

* Practical understanding of database design and management
* Advanced SQL concepts: `GROUP BY`, `ROLLUP`, `WINDOW FUNCTIONS`, `TO_DATE()`
* Real-world experience in data cleaning, analysis, and reporting
* Prepared for analytics and data science workflows
